# BShootr

Yet another boomer shooter.

A Raycasted first person shooter my own engine made via a minimum replication of ncurses. Made by NodeMixaholic.

## How to run

- Mac Arm64: `./bshootr`
- Linux/Other UNIX-likes: `make clean && make build && ./bshootr`

## Lore

The pillar god has created pillar monsters to attack humanity! Your goal is simple: Destroy all the pillar monsters. Good luck.
